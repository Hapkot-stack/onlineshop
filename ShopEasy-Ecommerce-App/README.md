## Project Overview

ShopEasy is a modern mobile e-commerce application prototype designed using Figma. The application enables users to browse products, view product details, add products to a cart, and manage their profiles through a user-friendly interface.

The design follows modern UI/UX principles with a focus on usability, accessibility, consistency, and mobile responsiveness.

## Features

- Hero / Welcome Page
- User Registration
- User Login
- Product Browsing
- Product Details View
- Shopping Cart
- User Profile Management
- Interactive Prototype Navigation
