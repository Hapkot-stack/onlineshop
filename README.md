## Project Overview

ShopEasy is a modern mobile e-commerce application prototype designed using Figma. The application provides users with a simple, attractive, and user-friendly shopping experience. Users can create an account, log in, browse products, view product details, add items to their shopping cart, and manage their profile.

The project demonstrates the application of modern UI/UX principles, wireframing, high-fidelity interface design, interactive prototyping, accessibility, and responsive mobile design.

## Instructions for Accessing the Project

1. Open the Figma project using the link above or the link provided in `prototype/figma-link.md`.
2. Review the User Persona, User Flow, Problem Statement, and Accessibility pages in the Figma file.
3. Review the Wireframes located in the `wireframes` folder.
4. Review the High-Fidelity UI designs located in the `high-fidelity-designs` folder.
5. Launch the interactive prototype in Figma by clicking the **Present (▶)** button.
